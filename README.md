# GitPush — GitHub Folder Uploader Chrome Extension

Удобное Chrome-расширение, которое позволяет загружать целые локальные папки в ваши репозитории GitHub в один коммит с использованием Git Data API.

## Возможности
- **Загрузка в один коммит:** Все файлы загружаются как отдельные бинарные/текстовые блобы (Git Blobs), после чего создается единое дерево, один коммит и обновляется указатель ветки. Это намного быстрее и чище, чем делать коммит на каждый файл.
- **Автоматическая загрузка репозиториев и веток:** Достаточно ввести токен — расширение само подгрузит список ваших репозиториев (с правами на запись) и веток.
- **Поддержка прокси из коробки:** Сетевые запросы выполняются стандартным браузерным `fetch()`, благодаря чему расширение автоматически наследует любые системные или браузерные настройки прокси (включая VPN и прокси-расширения).
- **Фильтрация мусора:** Автоматически отфильтровывает служебные папки `.git`, `node_modules` и пустые файлы при загрузке.
- **Безопасность данных:** Персональный токен доступа (PAT) хранится локально в браузере с помощью `chrome.storage.local` и никуда не передается, кроме официального GitHub API (`api.github.com`).

## Установка

1. Скачайте или клонируйте репозиторий в локальную папку (например, `C:\repo\ghub-chrome-ext`).
2. Откройте Google Chrome и перейдите по адресу `chrome://extensions/`.
3. В правом верхнем углу включите тумблер **Режим разработчика** (Developer mode).
4. Нажмите кнопку **Загрузить распакованное расширение** (Load unpacked) в левом верхнем углу.
5. Выберите папку проекта (`C:\repo\ghub-chrome-ext`), где лежит файл `manifest.json`.
6. Расширение появится в списке установленных. Закрепите его на панели инструментов Chrome для быстрого доступа.

## Как получить GitHub Personal Access Token (PAT)

1. Перейдите в [GitHub Settings -> Developer Settings -> Personal Access Tokens -> Tokens (classic)](https://github.com/settings/tokens).
2. Нажмите **Generate new token (classic)**.
3. Укажите название токена (например, `GitPush Chrome Extension`) и выберите область видимости (Scopes):
   - Отметьте флажок **`repo`** (дает полный доступ к управлению репозиториями).
4. Нажмите **Generate token** внизу страницы и скопируйте созданный токен.
5. Вставьте токен в расширение и нажмите **Войти**.

## Использование

1. Нажмите на иконку расширения.
2. Введите ваш Personal Access Token и нажмите **Войти**.
3. Выберите нужный репозиторий из выпадающего списка.
4. Выберите целевую ветку репозитория (например, `main` или `master`).
5. Нажмите кнопку **Выбрать папку** и выберите локальную папку с файлами на компьютере.
6. Введите сообщение коммита (или оставьте значение по умолчанию).
7. Нажмите **Загрузить на GitHub** и наблюдайте за ходом выполнения в консоли расширения.
8. После успешной загрузки нажмите кнопку **Открыть на GitHub**, чтобы просмотреть созданный коммит.
